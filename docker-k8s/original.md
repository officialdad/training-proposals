# DevOps: Docker & Kubernetes for Modern Application Deployment

**TRAINING PROPOSAL** — Crux Institute

- **Duration:** 2 Days · 16 Hours
- **Classification:** Digital Skills
- **Prepared for Review:** April 2026

---

## 01 Course Overview

This program is designed to provide participants with practical knowledge and hands-on experience in modern DevOps practices, focusing on containerisation using Docker and container orchestration using Kubernetes.

Participants will learn how to package applications into containers, manage container lifecycles, and deploy scalable applications in a Kubernetes environment. The program bridges the gap between development and operations by introducing automation, continuous deployment concepts, and infrastructure management.

Aligned with Industry 4.0 and cloud-native transformation, this program equips participants with essential skills required for modern software development, deployment, and system scalability.

## 04 Course Objectives

At the end of this program, participants will be able to:

1. Understand DevOps principles and containerisation concepts
2. Build and manage Docker containers
3. Create and manage container images
4. Deploy and manage applications using Kubernetes
5. Understand container orchestration and scaling
6. Apply DevOps practices in real-world environments

## 05 Learning Outcomes

Upon completion of the program, participants will be able to:

- Explain the role of Docker and Kubernetes in DevOps
- Build, run, and manage containerised applications
- Create Docker images and manage repositories
- Deploy applications in Kubernetes clusters
- Manage scaling, load balancing, and service exposure
- Implement basic DevOps workflows for deployment

## 07 Course Outline

*2-day programme · 16 hours of intensive, hands-on learning*

### Day 1: Docker & Containerisation (8 Hours)

**Hour 1 — Introduction to DevOps & Containers**
- Overview of DevOps principles
- Challenges in traditional deployment
- Introduction to containerisation
- Benefits of Docker in modern applications

**Hour 2 — Docker Fundamentals**
- Docker architecture and components
- Installing and setting up Docker
- Working with Docker CLI
- Understanding images and containers

**Hour 3 — Building Docker Images**
- Writing Dockerfiles
- Building custom images
- Image layers and optimisation
- Managing image versions

**Hour 4 — Container Management**
- Running and managing containers
- Networking in Docker
- Volume management and persistence
- Debugging containers

**Hour 5 — Docker Compose**
- Introduction to multi-container applications
- Writing docker-compose files
- Managing services and dependencies
- Running applications with Docker Compose

**Hour 6 — Docker Best Practices**
- Image optimisation
- Security considerations
- Resource management
- Version control for containers

**Hour 7 — Introduction to CI/CD Concepts**
- Continuous Integration & Continuous Deployment
- Role of Docker in CI/CD pipelines
- Basic automation workflows

**Hour 8 — Hands-On Lab (Docker Project)**
- Building and deploying a containerised application
- Testing and troubleshooting
- Review and discussion

### Day 2: Kubernetes & Orchestration (8 Hours)

**Hour 1 — Introduction to Kubernetes**
- What is Kubernetes and why it is needed
- Kubernetes architecture overview
- Key components (nodes, pods, clusters)

**Hour 2 — Kubernetes Core Concepts**
- Pods, deployments, and services
- Labels and selectors
- Namespace management

**Hour 3 — Deploying Applications in Kubernetes**
- Creating deployments
- Managing replicas and scaling
- Rolling updates and rollbacks

**Hour 4 — Networking in Kubernetes**
- Service types (ClusterIP, NodePort, LoadBalancer)
- Internal and external communication
- Ingress basics

**Hour 5 — Configuration & Storage**
- ConfigMaps and Secrets
- Persistent volumes and storage management
- Environment configuration

**Hour 6 — Monitoring & Troubleshooting**
- Logs and debugging tools
- Monitoring application health
- Troubleshooting deployment issues

**Hour 7 — Kubernetes Best Practices**
- Security fundamentals
- Resource management
- Scaling strategies
- High availability concepts

**Hour 8 — Hands-On Lab (Kubernetes Deployment Project)**
- Deploying a containerised application to Kubernetes
- Scaling and managing services
- Final review and Q&A

# Docker for Beginner

**COURSE OUTLINE** — Taming Tech Sdn Bhd

*Markdown equivalent of the supplied Taming Tech PDF proposal. Faithful to the original; no adjustments applied.*

- **Training Type:** Claimable Course · Public Training
- **Duration:** 3 days (21 hours)
- **Prerequisites:** Basic understanding of software development principles and command line usage.

## Course Description

This comprehensive course provides a foundational understanding of Docker, aimed at beginners eager to learn about containerization. Participants will explore the core architecture of Docker, understand the differences between images, containers, and Docker Compose, and gain hands-on experience in setting up and managing their Docker environment. Key skills include troubleshooting containers, implementing basic security measures, and leveraging Docker Compose for effective application deployment.

## Target Audience

- **Beginner Developers** who are new to containerization and Docker technology.
- **IT Professionals** who want to expand their knowledge in containerization.
- **Project Managers** who need a basic understanding of Docker for project oversight.
- **System Administrators** who are looking to improve their skills in managing Docker environments.

## Learning Objectives

- Understand the architecture of Docker, including images and containers.
- Be able to set up Docker and manage Dockerfiles effectively.
- Gain proficiency in using Docker Compose to manage multiple services including backend, database, and phpMyAdmin.
- Learn how to troubleshoot container issues and check logs.
- Understand basic security practices for Docker deployments.
- Be able to backup and restore container data securely.

## Course Outline

### Day 1: Introduction to Docker

**Module 1: Introduction to Containerization** *(1 hour)*
- Understand the basics of containerization and its advantages.
- Explore the differences between virtual machines and containers.

**Module 2: Installing Docker** *(1.5 hours)*
- Step-by-step installation of Docker on various platforms.
- Verify installation and perform initial configuration.

**Module 3: Docker Architecture** *(1.5 hours)*
- Learn about Docker architecture, including images, containers, and the Docker daemon.
- Understand the role of Docker Hub.

**Module 4: Working with Docker Images** *(1.5 hours)*
- Explore how to create and manage Docker images.
- Learn to use Docker commands for building and pulling images.

**Module 5: Hands-On Lab: Creating Your First Container** *(1.5 hours)*
- Hands-on activity to create a simple Docker container.
- Understand container lifecycle and management.

### Day 2: Managing Docker Containers

**Module 6: Docker Logging and Troubleshooting** *(1.5 hours)*
- Learn how to check logs and troubleshoot common container issues.
- Understand how to configure automatic restarts.

**Module 7: Docker Security Basics** *(1.5 hours)*
- Explore basic security practices for Docker containers and images.
- Learn how to secure Docker deployments.

**Module 8: Docker Compose Fundamentals** *(1.5 hours)*
- Understand the purpose of Docker Compose.
- Create and manage multi-container applications using Docker Compose.

**Module 9: Hands-On Lab: Docker Compose** *(1.5 hours)*
- Practical session on using Docker Compose to set up a multi-service application.
- Collaborate on deploying a simple app using Docker Compose.

### Day 3: Advanced Topics and Best Practices

**Module 10: Backup and Restore Procedures** *(1.5 hours)*
- Understand strategies for backing up and restoring container data.
- Learn best practices for data management.

**Module 11: Troubleshooting and Maintenance** *(1.5 hours)*
- Explore advanced troubleshooting techniques for complex container issues.
- Understand maintenance practices for Docker environments.

**Module 12: Final Project: Dockerize Your Application** *(2 hours)*
- Participants will work on a final project to dockerize an application of their choice.
- Present solutions and troubleshoot any issues encountered.

**Module 13: Course Wrap-Up and Q&A** *(1 hour)*
- Recap of key concepts learned throughout the course.
- Open floor for questions and additional discussions.

## Methodology

The course employs a blend of instructional techniques including lectures, demonstrations, and hands-on labs to facilitate learning. Participants will engage in practical exercises, allowing them to apply concepts immediately and solidify their understanding. The use of real-world scenarios and case studies helps participants relate to Docker's applications in various contexts.

## Expected Outcome

- Participants will be able to dockerize their own applications or systems.
- Participants will be able to deploy Docker containers on their own servers.
- Participants will be able to backup and restore container data securely.
- Participants will be able to troubleshoot container issues confidently.
- Participants will be able to use Docker Compose effectively for managing multi-container applications.
- Participants will not fear restarting or redeploying production containers.

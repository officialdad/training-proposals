# COURSE OUTLINE

| Field | Details |
|-------|---------|
| **PROGRAMME TITLE** | Website 101: Build & Deploy Your First Website with AI |
| **OVERVIEW** | This beginner-friendly programme introduces non-technical participants to the fundamentals of how websites work and how the internet delivers them. Participants will gain a foundational understanding of web technologies (HTML, CSS, JavaScript), how the internet works behind the scenes (domain names, servers, databases), and modern hosting platforms. Through guided hands-on practice, participants will use AI tools of their choice to build a personal portfolio website and publish it live on Cloudflare Pages. This course is designed as the first module in a step-by-step learning series, preparing participants for advanced topics in dynamic web development. |
| **LEARNING OUTCOME** | By the end of the session, the participants will be able to: |
| | a) **Explain** how websites work — DNS, servers, databases |
| | b) **Identify** the three building blocks — HTML, CSS, JavaScript |
| | c) **Use** AI tools to build a personal portfolio website |
| | d) **Publish** a website to the internet and verify it is live |
| **DURATION** (days) | 1 day (4.5 hours / 270 minutes) |
| **TARGET PARTICIPANTS** | Non-IT professionals, executives, administrators, entrepreneurs, and any individual with no prior web development experience who wishes to understand and apply basic website creation skills. Participants must have basic computer literacy (able to use a web browser and manage files). |

## LEARNING METHODOLOGIES

| Methodology | Description |
|-------------|-------------|
| Lecture & Visual Presentation | Trainer-led slides covering internet concepts (Module 1), building blocks (Module 2), and the AI approach (Module 3) |
| Live Demonstration | Trainer demonstrates DNS lookup live (Module 1) and walks through HTML, CSS, and JavaScript file creation (Module 2) |
| Guided Hands-On Practice | Participants create HTML, CSS, and JavaScript files (Module 2); build a portfolio with AI and deploy it (Modules 3–4) |
| AI-Assisted Learning | Participants use their choice of AI tool — ChatGPT, Claude, Gemini, or Grok — to generate and refine portfolio code |
| Peer Review & Sharing | Participants review each other's local portfolio files before deployment (Module 3) and share live URLs after deployment (Module 4) |
| Game-Based Quiz | Final Quiz on fuiz.org recaps all four learning outcomes in a competitive, interactive format (Module 4) |
| Q&A and Reflection | Structured Q&A at the end of each module; Learning Outcomes Achieved review at course close |

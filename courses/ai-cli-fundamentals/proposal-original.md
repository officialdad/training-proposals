## 01 Course Overview

Most executives meet AI through a browser tab — Gemini, ChatGPT, or Claude on the web. That works, but it traps users in click-by-click conversations, locks files inside copy-paste, and turns every task into a fresh start. This programme is built for executives who have outgrown that pattern and want a tighter, faster way of working with AI.

Across one intensive day, participants migrate from web-based AI chat to Gemini CLI on their own laptop. They learn how AI has fundamentally changed the layer at which we instruct computers, install the free-tier Gemini CLI, master structured prompting, bring their own files into the conversation, and automate a real repetitive task from their own role. By the end of the day, every participant leaves with at least one workflow they actually use at work — now driven from the command line.

## 03 Entry Requirements

- Familiarity with ChatGPT, Gemini, or Claude on the web (used the chat interface at least a few times)
- Basic computer literacy and comfort with a personal laptop
- Willingness to install free software on a personal or work laptop (admin rights required)
- A Google account (free) for Gemini CLI sign-in

## 04 Course Objectives

At the end of this programme, participants will be able to:

1. **Describe** how AI shifts the layer of computer instructions from clicks and code to natural language, and where chat, CLI, and API sit on that stack
2. **Configure** Gemini CLI on a personal laptop using the free tier
3. **Compare** browser-based AI chat with CLI-driven AI workflows and pick the right tool for each task
4. **Apply** structured prompting techniques (role, context, format, examples) for repeatable executive tasks
5. **Automate** a real-world repetitive workflow from their own role (weekly reports, vendor follow-ups, meeting summaries) using Gemini CLI commands and saved prompts
6. **Integrate** Gemini CLI with local files (PDFs, spreadsheets, notes) for context-aware outputs
7. **Evaluate** AI output for accuracy, bias, and confidentiality before sharing

## 05 Learning Outcomes

Upon completion of the programme, participants will be able to:

- Explain in plain language how a Large Language Model takes instructions, and why moving from web chat to CLI changes the experience
- Install, authenticate, and launch Gemini CLI on Windows or macOS using the free tier
- Choose deliberately between web chat, CLI, and (in future) API for any given task
- Write structured, reusable prompts that produce reliable output the first time
- Automate at least one of their own repetitive workplace tasks end-to-end with Gemini CLI
- Bring local files into the conversation safely, applying a confidentiality checklist before any data leaves the laptop
- Spot common AI failure modes — hallucination, bias, stale context — and apply a personal quality-control habit before sharing output

## 07 Course Outline

*1-day programme · 8 hours of intensive, hands-on learning*

**Hour 1 — AI Fundamentals: The Instruction Layer Has Changed**
- How computers were instructed before AI: clicks, menus, code, macros
- What a Large Language Model actually is, in plain language (predicts next tokens from context)
- The three modern interaction layers: chat (web), CLI (terminal), API (programmatic)
- Why moving up the stack saves time for executives — fewer clicks, more outcomes

**Hour 2 — Web AI vs CLI AI: Two Doors to the Same Model**
- Quick tour of Gemini, ChatGPT, and Claude on the web — strengths and limits
- What changes when the same model is on the command line: speed, files, scripting, privacy
- Cost lens: web subscriptions vs CLI free-tier quotas
- When the browser is still the right answer (and when it isn't)

**Hour 3 — Setting Up Gemini CLI on Your Laptop**
- Installing Node.js and Gemini CLI on Windows and macOS
- Signing in with a Google account; understanding free-tier limits
- First conversation in the terminal: the `gemini` command and the basics
- Troubleshooting common install issues (PATH, proxy, permissions)

**Hour 4 — Prompting Foundations: From Vague to Reliable**
- Anatomy of a strong prompt: role + context + task + format + examples
- Why CLI prompts can be longer and more structured than web chat
- Using multi-line prompts and saved prompt files
- Live walkthrough: rewriting the same task three ways and comparing outputs

**Hour 5 — Bringing Your Own Files into the Conversation**
- Passing PDFs, spreadsheets, and notes to Gemini CLI as context
- Summarising meeting transcripts and long reports in one command
- Extracting structured data from messy documents
- Confidentiality checklist: what should never leave your laptop

**Hour 6 — Automating Repetitive Work**
- Identifying repeatable tasks in an executive workday
- Saving prompts as reusable templates and shell aliases
- Chaining commands: read file → ask Gemini → save result
- Worked example: weekly status update generated from raw notes

**Hour 7 — Your Personal AI Toolkit**
- Organising prompts in a personal `prompts/` folder
- Quality-control habits: spot-check, fact-check, redact before sharing
- Bias, hallucination, and confidentiality risks executives must own
- Sharing prompts across a team without sharing data

**Hour 8 — Hands-On Lab: Automate One Real Workflow**
- Each participant picks one real repetitive task from their own role
- Build the prompt, wire in their files, save it as a reusable script
- Trainer walks the room providing 1:1 feedback
- Group share-out: what worked, what surprised them, Monday next step
